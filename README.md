# BarberosFree
Barberos Free nace a partir de la necesidad, en El Salvador existen muchos barberos que trabajan de manera independiente, esta es una herramienta que unen a todos los barberos para poder tenerlos a todos en una plataforma, ayudando a que se agenden perfectamente, que tengan una gestión de reservas, catalogos de cortes, calificaciones de barberos para saber cual es el más confiable y saber que tan bueno es el servicio que ofrece a traves de calicaciones y comentarios.
